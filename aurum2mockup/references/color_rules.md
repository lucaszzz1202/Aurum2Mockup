# 色彩规范

# **品牌色**

作为服务于toB业务的产品设计，我们希望能够继续沿用麦当劳的品牌色，营造熟悉、轻松、愉快的氛围。因此我们将toB业务的品牌主色选定为 #FFBC0D，并对主色进行了明度衍生。

- 品牌色，麦当劳金：#FFBC0D
- 明度衍生，从明到暗为：
    - Gold-1:#FFF8E6
    - Gold-2:#FFF2CB
    - Gold-3:#FFBC0D（品牌主色）
    - Gold-4:#B37400

# **辅助色**

保留麦当劳红作为辅助色，并增加了极客蓝、雾霭灰，扩展了色彩使用范围，满足业务线的色彩需要，从中体现出专业、科技感的色彩感觉。辅助色主要用于图标、标签类色彩点缀。

- 麦当劳红：
    - Red-1:#DB0007
    - Red-2:#FFE9E6
- 极客蓝：
    - Blue-1:#4880FF
    - Blue-2:#EBF3FF
- 清新绿：
    - Green-1:#4EBE77
    - Green-2:#E6F6EA

在色彩使用上，辅助色使用优先级：麦当劳红＞ 极客蓝 ＞ 清新绿 。同时要注意控制辅助色在整个页面中的使用比例，需要拥有充足的呼吸空间。

整体色彩使用比例：

金色20%，红色3%，其他7%，留白70%

当大面积使用品牌黄&红色时，请将透明度降低至50%

# **功能色**

功能色为界面设计中的特殊场景色，主要为成功、失败、警告、链接、通知等。为避免色彩滥用，同时方便使用，我们继续沿用品牌色，并结合使用场景进行功能划分。

- 成功：#4EBE77
- 失败：#DB0007
- 警告：#FFBC0D
- 链接：#4880FF
- 通知：#EBF3FF

# **中性色**

中性色主要用于文字、背景、边框、分割线等，因此需要考虑文字及背景搭配的视觉效果，结合 WCAG2.0（Web 内容无障碍指南），保证常用文字清晰易读。

- 文字：
    - 文字/标题/价格：Gray-5:#222222
    - 正文：Gray-4:#666666
    - 描述：Gray-3:#999999
    - 占位提示：Gray-2:#CCCCCC
    - 白色：White：#FFFFFF
- 背景/蒙层：
    - 页面背景：Gray-1:#F6F6F6
    - 侧导航背景：Gray-6:#1B1B1B
    - 黑色：Black：#000000-蒙层使用透明度70%
- 分割线：
    - 最弱分割线：Line-1：#000000-6%
    - 常规分割线：Line-2：#000000-12%
    - 强调分割线：Line-3：#000000-24%
